About Me

I’ve been fascinated by computers and their capabilities since my first Java class in 2018 at SVC. The challenge of writing code that behaves exactly as intended hooked me—each assignment felt like a puzzle to solve. The blend of problem-solving and creativity was incredibly rewarding.

When I discovered web development, I knew I’d found my path.It combines the logic of programming with the excitement of building interactive, visual experiences that anyone on the internet can use.

Creating applications that are used and appreciated inspires me to continue creating. Today I’m eager to contribute as a web developer, applying my skills in JavaScript, HTML/CSS, and React to create user-focused, performant web applications.
